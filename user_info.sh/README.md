## user_info.sh

## Description
- user_info.sh is a Bash shell script that retrieves and displays system account information for a specified user. 
It reads data from the /etc/passwd file and calculates the size of the userâ€™s home directory.

## Skills Demonstrated
- Bash shell scripting and variables
- Command-line text processing
- Reading and parsing system files
- Working with user and file system information
- Basic calculations and output formatting

## Features
- Displays username, UID, GID, and comment field
- Shows home directory path and login shell
- Calculates and displays home directory size (KB)
- Accepts a username as a command-line argument

## Run
./user_info.sh username
