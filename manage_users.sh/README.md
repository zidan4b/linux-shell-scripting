## manage_users.sh

## Description
- manage_users.sh is a Bash shell script that provides an interactive, menu-driven interface for Linux user administration. 
The script runs in a continuous while loop and uses a case statement to execute user-selected operations until the program exits.

## Skills Demonstrated
- Bash shell scripting (functions, variables, exit status)
- Control flow: while loop and case menu
- Linux user and group management
- Reading user input and basic error handling
- Working with system files and commands

## Features
- Create and delete user accounts
- Modify primary and supplementary groups
- Change login shells
- Set account expiration dates
- Requires sudo privileges for administrative actions

**Run**
chmod +x manage_users.sh
./manage_users.sh
